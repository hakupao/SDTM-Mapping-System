# 🚀 VAPORCONE 后端启动指南

## 📁 重构后的清洁架构

经过重构，后端代码已从原来的单个1189行文件优化为模块化架构：

```
backend/
├── 🚀 app.py                  # 主应用入口 (~150行)
├── ⚙️ config.py               # 配置管理
├── 🧪 test_new_architecture.py # 架构测试
├── 🚀 start.bat               # 启动脚本
├── 📖 README_重构架构.md       # 详细架构文档
├── api/                       # API蓝图模块
├── services/                  # 业务逻辑服务
├── websockets/                # WebSocket处理
├── models/                    # 数据模型 (预留)
└── utils/                     # 工具函数 (预留)
```

## 🚀 快速启动

### 方法1：使用启动脚本 (推荐)
```bash
cd vaporcone-frontend/backend
start.bat
```

### 方法2：手动启动
```bash
cd vaporcone-frontend/backend
pip install -r ../../requirements.txt
python test_new_architecture.py  # 测试架构
python app.py                    # 启动服务器
```
